# 2D Graphics Editor

## Mini Project

A simple 2D Graphics Editor developed in C that allows users to create and manage basic geometric shapes on a graphics canvas.

## Features

- Draw Rectangles
- Draw Lines
- Draw Triangles
- Draw Circles
- Store multiple graphical objects
- Simple object management system
- Console-based implementation in C

## Technologies Used

- C Programming Language
- Standard C Libraries
  - stdio.h
  - stdlib.h
  - ctype.h

## Project Structure

- `main.c` – Main source code
- `editor.exe` – Executable program
- `prompt.txt` – Project documentation and requirements

## Constants

- Canvas Width: 80
- Canvas Height: 25
- Maximum Objects: 100

## Supported Shapes

1. Rectangle
2. Line
3. Triangle
4. Circle

## How to Compile

```bash
gcc main.c -o editor
```

## How to Run

```bash
./editor
```

For Windows:

```bash
editor.exe
```

## Objective

The objective of this project is to implement a basic 2D graphics editor using C programming concepts such as structures, enumerations, arrays, and file handling.
